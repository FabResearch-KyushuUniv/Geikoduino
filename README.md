# Geikoduino

## Geikoduinoとは
Geikoduinoは九大大橋キャンパスの形をしたarduinoと同じ機能を持った電子基盤です。
ご当地arduinoです。
ミリングマシンの勉強にもなります。
作り方の詳細を書いていますので、もし作りたい方がいれば↓こちらを参照してください。
